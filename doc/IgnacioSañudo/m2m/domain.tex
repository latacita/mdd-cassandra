\chapterheader{Model to text}{Model to text}
\label{chap:domain}

Este capítulo describe el primer paso del proceso de transformación de acuerdo al proceso de desarrollo dirigido por modelos. El principal objetivo de esta fase es la transformación de un modelo definido en UML  a un modelo en Cassandra. La primera fase consiste en la definición de un metamodelo que defina el modelado en Cassandra. En segundo lugar debemos  definir las reglas de transformación para convertir un modelo UML a un modelo en Cassandra utilizando para ello el lenguaje ETL.
