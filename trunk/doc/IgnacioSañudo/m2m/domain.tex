\chapterheader{Transformación modelo a modelo}{Transformación modelo a modelo}
\label{chap:m2m}

Este capítulo describe el primer paso del proceso de transformación de acuerdo al proceso de desarrollo dirigido por modelos. El principal objetivo de esta fase es la transformación de un modelo definido en UML  a un modelo en Cassandra. La primera fase consiste en la definición de un meta-modelo que defina el modelado en Cassandra. En segundo lugar debemos definir las reglas de transformación para convertir un modelo UML a un modelo en Cassandra utilizando para ello el lenguaje Epsilon Transformation Language (ETL). A continuación se presenta un caso de estudio en el cual se van a aplicar las reglas de transformación ETL que se han definido. Finalmente el sumario con las conclusiones de este capítulo.
